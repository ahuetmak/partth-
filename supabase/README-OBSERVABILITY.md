# Observability para Edge Functions

El archivo `observability.json` define la configuración de logs y traces para las Edge Functions.

## Aplicar en Supabase Cloud

1. Ve a **Supabase Dashboard** → tu proyecto → **Project Settings** → **Edge Functions**
2. Busca la sección **Observability** o **Logging**
3. Activa logs y traces según la configuración en `observability.json`

Si tu plan incluye configuración avanzada, aplica los valores manualmente.

## Valores actuales

- **Observability**: habilitada
- **Logs**: habilitados, persistencia activa, invocation_logs activo
- **Traces**: habilitados, persistencia activa
